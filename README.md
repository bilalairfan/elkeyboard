# elkeyboard

A 3x3 macropad I will primarily use for CADding and school work.

## Features
- 8 keys
- a rotary encoder
- 128x32 OLED display
- LEDs under each key

## Firmware
My layout is going to be based on my shortcuts for my cad software (onshape).
The bottom line is going to be general purpose shortcuts while the top
and middle are more specific to working on onshape. This is primarily
sketch constraints as I tend to forget my binds when I'm working. 
Additionally, I have an encoder binded to my volume as I tend to change my volume a lot between days and tasks.

## BOM:
- 8x cherry mx switches
- 8x keycaps
- 1x ec11 rotary encoder
- 9x sk6812 mini e
- 9x 1N4148 diodes
- 1x OLED .91" 128x32
- 1x XIAO RP2040
- 1x case
