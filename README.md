# elkeyboard

My layout is going to be based on my shortcuts for my cad software (onshape).
The bottom line is going to be general purpose shortcuts while the top
and middle are more specific to working on onshape. This is primarily
sketch constraints as I tend to forget my binds when I'm working. 
Additionally, I have an encoder binded to my volume.
